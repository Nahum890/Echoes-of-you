# Level Kit — Echoes of You

Prefabs editables para montar y retocar niveles sin depender solo del builder.

## Generar / actualizar

En Unity: **Echoes of You → Production → Export Level Kit Prefabs**

## Contenido

| Prefab | Uso |
|--------|-----|
| `EchoPathHint.prefab` | Waypoints de luz para rutas de eco (edita posiciones hijas o el array en el componente) |
| `PressurePlate_Eco.prefab` | Placa para eco / proyección F |
| `PressurePlate_Player.prefab` | Placa del jugador |
| `TutorialTrigger.prefab` | Zona de tutorial |

Arrastra estos prefabs bajo `--- MECHANICS ---` en la escena del nivel.
